# Timer-App
A simple Timer app using Android Architecture Components and the MVVM design pattern.


![Timer App](https://gabriel-tanner.firebaseapp.com/images/timer-work-running.png)

### Features
This app functions as a simple pomodoro timer where you can change the time of your sessions and get reminders when a session is over.
